# Contributing to Simple Web Calculator

Thanks for your interest in contributing! Contributions are welcome and appreciated.

Guidelines

- Bug reports: Open an issue describing the problem, steps to reproduce, and expected vs actual behavior.
- Feature requests: Open an issue with a clear description and use cases.
- Pull requests:
  - Fork the repository and create a feature branch (e.g., `fix/clear-button`, `feat/history`).
  - Keep changes focused and include tests or screenshots when applicable.
  - Follow the existing code style (plain JavaScript, simple CSS).
  - Write descriptive commit messages.
  - Run the app locally to verify behavior before opening a PR.

Code of conduct

Be respectful and inclusive. By participating you agree to follow the project's code of conduct.

License

By contributing, you agree that your contributions will be licensed under the project's MIT license.
